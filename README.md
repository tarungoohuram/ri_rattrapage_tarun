# ri_rattrapage_tarun
Installation

The package need to be install in your catkin workspace.

cd catkin_ws/src
git clone https://github.com/tarungoohuram/ri_rattrapage_tarun.git
catkin build
cd ..
source devel/setup.bash

Execution

source devel/setup.bash
roslaunch ri_rattrapage_tarun vizualize_urdf.launch
